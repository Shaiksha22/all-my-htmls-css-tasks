About Lists

In html we have types of lists:-

  1. ordered lists
  2. unordered lists
  3. description lists

ORDERED LISTS:-

Ordered lists are used to creating lists with the numbers, uppercase letters, lowercase letters or roman numeral tag used for creating the ordered list is < ol > < li > </ li > </ ol >
ex:- month names in ordered lists
  1. june
  2. july
  3. aug
  4. sept
  5. oct
  6. nov
  7. dec

UNORDERED LISTS:-

We use unordered lists to group items having no order for creating the ordered list is < ul > < li > </ li > </ ul >
ex:- month names in ordered lists
  • jan
  • feb
  • march
  • april
  • may

Description LISTS:-

Which defines the description list and tag which defines the terms in the description list < dl > < dt > < /dt > <dd > </dd > <dl >
